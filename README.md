# Password Generator (C++ CGI)

## Description
This project is a secure password generator built using HTML, CSS, JavaScript,
and a C++ backend implemented using CGI.

## Features
- Password length slider
- Character selection (lowercase, uppercase, digits, special characters)
- Password strength validation
- Password history
- C++ backend using CGI (Apache/XAMPP)

## Technologies Used
- C++
- HTML
- CSS
- JavaScript
- Apache (XAMPP)

## How It Works
The frontend sends user input to a C++ CGI program using JavaScript.
The Apache server executes the CGI program and returns the generated
password and strength.
